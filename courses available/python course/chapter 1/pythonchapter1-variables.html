Chapter 1 - Work with variables

Variables are the smallest measurement in programming. So how do we work with these tiny blocks to build a full program with Python?

Examples

Some examples of variables

Declaring variables

greetings = "Hello World"

Presenting variables

greetings = "Hello World!" print(greetings)

Printing multiple variables

z = 18 q = 12 x = 37 print(z) print(q) print(x)

Adding variables

z = 17 q = 20 print(z + q) greetings1 = "Hello " greetings2 = "World!" print(greetings1 + greetings2)
