Chapter 1 - Working with variables

Variables are the basic measurement in programming. So how do we work with these tiny blocks to build a full program with Python?

Examples

Some examples of variables

Declaring variables

greetings = "Hello World"

Presenting variables

greetings = "Hello World!"
print(greetings)

Printing multiple variables

z = 18
q = 12
x = 37
print(z)
print(q)
print(x)

Adding variables (concatenate)

z = 17
q = 20
print(z + q)
# Expected output: 37
greetings1 = "Hello "
greetings2 = "World!"
print(greetings1 + greetings2)
# Expected output: Hello World!
